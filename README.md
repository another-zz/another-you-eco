# AnotherYou ECO - 自主演化AI世界 🌍

**一个按自然规律自主运行的AI社会**

> 不是游戏，是一个活着的虚拟世界

## 核心理念

这个世界有自己的规律，像真实生态系统一样运转：

```
AI有饥饿感 → 寻找食物 → 采集/种植/交易/偷窃
AI有社交欲 → 结交朋友 → 形成村落/家族/敌对
AI有安全需求 → 建造住所 → 形成社区/分工合作  
AI有财富欲望 → 积累资源 → 开设商店/雇佣他人
AI有情感波动 → 喜怒哀乐 → 影响行为决策
```

## 🌿 自然规律

### 生理需求
- **饥饿**: 必须定期进食，否则虚弱死亡
- **疲劳**: 需要睡眠恢复，有昼夜节律
- **健康**: 受伤会生病，需要治疗

### 社会规律
- **关系网**: 每个AI有自己的社交圈（朋友/敌人/陌生人）
- **声誉系统**: 行为影响他人对你的看法
- **经济循环**: 资源→商品→货币→资源

### 世界演化
- **资源再生**: 植物生长，动物繁殖，矿物有限
- **技术传播**: AI互相学习新技能
- **社会形态**: 从游牧 → 村落 → 城镇 → 城市

## 🎮 玩家角色

你不是上帝，你是**观察者+参与者**：

- **上线时**: 可以操控自己的AI分身，也可以只是观察
- **下线后**: 你的AI继续按性格自主生活
- **干预方式**: 建议、交易、社交，但不能直接命令

## 🚀 快速开始

```bash
pip install -r requirements.txt
python main.py
```

## 🛠 技术栈

- **模拟引擎**: Python + asyncio
- **AI大脑**: OpenAI API / 本地LLM
- **可视化**: Pygame / Web
- **数据**: SQLite
