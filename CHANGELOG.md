# AnotherYou ECO v0.2 更新日志

## 🚀 新功能

### 1. AI大脑 v2.0
- **向量记忆系统** - 借鉴斯坦福Generative Agents
  - 记忆流存储所有经历
  - 向量相似度检索相关记忆
  - 反思机制形成高层次洞察
  
- **LLM智能决策** - 借鉴AutoGPT
  - 观察→记忆检索→规划→行动循环
  - 自然语言理解复杂情境
  - 决策理由透明化

- **智能对话** - 基于关系和记忆生成对话
  - 考虑双方关系亲疏
  - 检索相关记忆
  - 符合性格特点

### 2. 动态经济系统
- **供需定价** - 市场价格随供需波动
- **自动撮合** - AI自动买卖匹配
- **价格报告** - 实时查看市场趋势

### 3. 农场系统
- **种植** - 小麦、蔬菜、水果
- **生长** - 受天气和土壤影响
- **收获** - 成熟后自动收获

## 🔧 技术升级

| 组件 | v0.1 | v0.2 |
|------|------|------|
| 记忆 | 简单列表 | 向量存储+反思 |
| 决策 | 规则权重 | LLM+规则fallback |
| 对话 | 预设语句 | LLM生成 |
| 经济 | 框架 | 供需驱动 |
| 农场 | 无 | 完整种植循环 |

## 📝 配置

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 配置API密钥
cp .env.example .env
# 编辑 .env 填入 OPENAI_API_KEY

# 3. 运行
python main.py
```
